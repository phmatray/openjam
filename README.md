## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

Read the whitepaper of the project on [GitBook](https://whitepaper.openjam.eu) or in [PDF](http://bit.ly/OpenJam-WP).

#### üñê Requirements

Node:

- NodeJS >= 10.x
- NPM >= 6.x

#### ‚è≥ Installation

```bash
git clone https://github.com/openjam-eu/openjam.git
```

**We recommend to switch on the dev branch to get the latest version of OpenJam**.
Some breaking changes might happen, new releases are shipped every week (more or less) to fix/enhance the product.

#### üöÄ Start your project

```bash
cd openjam
npm i
npm run dev
```

Congratulations, you made it! Enjoy üéâ

## Build With

### MERN

**MERN** is a set of Open Source components that together, provide an end-to-end framework for building dynamic web applications; starting from the top (code running in the browser) to the bottom (database). The stack is made up of:

- [**R**eact](https://reactjs.org) / [**R**edux](https://redux.js.org) : Front-end web app framework; runs your JavaScript code in the user's browser, allowing your application UI to be dynamic
- [**E**xpress](http://expressjs.com/) : Back-end web application framework running on top of Node.js
- [**N**ode.js](https://nodejs.org/en/) : JavaScript runtime environment ‚Äì lets you implement your application back-end in JavaScript
- [**M**ongoDB](https://www.mongodb.com/) : Document database ‚Äì used by your back-end application to store its data as JSON (JavaScript Object Notation) documents

### Styled-components

Utilising tagged template literals (a recent addition to JavaScript) and the power of CSS, styled-components allows you to write actual CSS code to style your components. It also removes the mapping between components and styles ‚Äì using components as a low-level styling construct could not be easier!

[Styled-components website](https://www.styled-components.com)

### Fractal

Fractal‚Ää is a react app structure for infinite scale so natural that on-boarding new frontend developers, became a piece of cake.

[Read Shivek Khurana\'s blog post about Fractal](https://hackernoon.com/fractal-a-react-app-structure-for-infinite-scale-4dab943092af)
